Explication du FormatLigne 1 : # Nombre de nœuds suivi du nombre total de nœuds dans le graphe (ici, 5).Section "Définition des types de nœuds" :Chaque ligne commence par l'ID d'un nœud, suivi de son type (0, 1 ou 2).Par exemple, 0 0 signifie que le nœud avec l'ID 0 est de type 0 (hub).Section "Liste d'adjacence avec attributs" :Chaque ligne décrit une arête (connexion) entre deux nœuds.Les valeurs sont séparées par des espaces.NoeudSource: L'ID du nœud de départ de l'arête.NoeudDestination: L'ID du nœud d'arrivée de l'arête.Distance: La distance de l'arête.BaseTime: Le temps de base pour parcourir l'arête.Cout: Le coût de l'arête.RoadType: Le type de route (par exemple, 1 pour autoroute, 2 pour route nationale, etc.).Reliability: La fiabilité de l'arête (par exemple, un nombre entre 0 et 1).Restrictions: Les restrictions de l'arête.TimeSlotStart1 TimeSlotEnd1 TimeSlotMultiplier1 TimeSlotStart2 TimeSlotEnd2 TimeSlotMultiplier2 ...:  Les tranches horaires (optionnelles) et leurs multiplicateurs. Par exemple,  8 17 1.2 signifie que de 8h à 17h, le temps de parcours est multiplié par 1.2.Comment Utiliser ce FichierEnregistrez le contenu ci-dessus dans un fichier nommé data/network.txt (ou tout autre nom que vous souhaitez utiliser).Assurez-vous que le fichier data/network.txt se trouve dans le même répertoire que votre exécutable C.Exécutez votre programme C.
